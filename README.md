# Software-Engineering-Project
The Soonerview Project 

Highlights
- Bookmarking
- Map Display
- colored coded district voting map
- Accessing votes
- Easier to understand what bills were voted for

Overview
While many pay attention to federal politics, state politics generally falls under the radar. In a safe state such as Oklahoma, this is even more so. Inspired by Voteview.com, SoonerView provides a recources for Oklahomans to explore votes in the Oklahoma Legislature (including committee votes).

Instructions:
1. head to soonerview.org
2. create a soonerview account
3. click on any bill to view it and bookmark it with the icon
4. ai overview is there to summarize the bill, the bill itself is there if wish to be red in full.
5. color codes from voting: red = republican yea, lighter red = republican Nay, dark blue = democratic Yea, light blue = Democratic Nay, Grey = no vote/absent.
